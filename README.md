# JavaScript-2
Course Assignment, Social media front-end application


📖 Description

A front-end social media application built with JavaScript. The app connects to Noroff’s API and allows users to register, log in, and interact with posts and other users.

Features:
- Register and log in users
- View posts (feed)
- Create, edit, and delete posts (CRUD)
- View profiles
- Follow / unfollow users
- Comment and react to posts
